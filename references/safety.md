# Safety

Use this file whenever the user wants current trade or investment guidance.

## Requirements

- Use fresh market data for current analysis.
- State the exact symbol, timeframe, and range used.
- State assumptions when the screenshot or prompt is incomplete.
- Do not fabricate data or imply certainty.

## Recommendation guardrails

- Prefer scenario-based advice over one-way certainty.
- Include invalidation and risk in every actionable idea.
- If the chart is low quality or range-bound, say that waiting is valid.
- Do not present the output as personalized financial advice.

## Good phrasing

- "Bull case improves if price holds above..."
- "Bear case strengthens on a close below..."
- "If this breakout lacks follow-through, treat it as a failed breakout risk."

## Bad phrasing

- "This will go up."
- "Guaranteed breakout."
- "Full size position is safe."
